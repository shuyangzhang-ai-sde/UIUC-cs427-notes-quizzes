What Is a Code Smell?
Design DecayWarning SignNot Always a Bug
A code smell is a warning sign that the code's internal structure is becoming harder to evolve. It does not always break behavior immediately, but it raises the risk and cost of future changes.
OptionIs it a code smell?Why
Design starts to decayYesSmells point to maintainability and design quality issues.
Syntax errorNoSyntax errors are immediate compile-time defects, not smells.
Optimized codeNoOptimization is not itself a warning sign.
Security vulnerabilityNo (different category)Security issues are vulnerabilities, not smell classification.
Exam Tip
The best definition is: a clear sign that code design is starting to decay.
Duplicated Code and Refactoring
Duplicated CodeExtract MethodSingle Source of Truth
When similar logic appears in multiple places, maintenance becomes error-prone. The most common first fix is Extract Method: move the repeated fragment into one named method and call it from each location.
RefactoringBest for duplicated code?Reason
Extract MethodYesConsolidates repeated logic into one reusable implementation.
Introduce Parameter ObjectSometimesSolves long parameter lists/data clumps, not primary duplication cleanup.
Inline MethodUsually noCan increase duplication by spreading logic back into callers.
Rename VariableNoImproves naming only; does not remove duplicated logic.
Long Method Smell
Long MethodComplexityReadability
  • Long methods mix too many responsibilities and hide intent.
  • They increase cognitive load and slow down bug localization.
  • They are harder to unit test because setup and branching become tangled.
  • Refactor by extracting cohesive chunks with names that express purpose.
True/False Focus
The statement is True: a Long Method smell is problematic because complexity makes code harder to understand and maintain.
Quick Review
  • Code smell = design quality warning, not necessarily immediate failure.
  • Duplicated Code is commonly addressed with Extract Method.
  • Long Method is a maintainability risk and should be decomposed.
